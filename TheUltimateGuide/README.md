# The Ultimate Guide to Peak Productivity

This is not a professional project by any means...

This project was created as part of the “Challenge Yourself” in the “Wrap Up” of the Layout and Style section from Apple’s beginner “Develop in Swift” tutorials. It’s based on concepts learned in the tutorial rather than a step-by-step implementation. Check the section here: https://developer.apple.com/tutorials/develop-in-swift/layout-and-style-conclusion

Skills developed in this module:

- Reducing code using type inference
- Debugging layout issues using borders
- Adjusting the sizes of elements using frames
- Using three types of stacks — VStack, HStack, and ZStack — to create a complex interface
- Controlling the placement of content in a stack with spacers
- Navigating between views using a paged tab view
- Custom color sets in the asset catalog

PS: I also used AI to add some final styling in two views—check the prompts in [Prompts](./prompts/). The main content, though, was coded like it was in the Stone Age.

PS2: Some modifiers were also suggested by AI to give the views a more modern look.
